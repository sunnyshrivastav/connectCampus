# 📚 ExamNotes: AI-Powered Learning & Assessment Platform

ExamNotes ek advanced platform hai jo AI ki madad se students ki learning aur assessments ko manage karta hai.

## 🚀 Features
* **AI Analysis:** Har test aur assessment ka deep analysis.
* **Lab Registration:** Medical labs ko register karne aur tests analyze karne ka system.
* **Smart Comparison:** Tests ki accuracy aur prices ka comparison.
* **End-to-End Workflow:** Booking se lekar payment aur reports tak ka poora process.

## 🛠️ Tech Stack
* **Frontend:** React.js (Client folder)
* **Backend:** Node.js & Express (Server folder)
* **Database:** MongoDB

## 💻 How to Run
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies:
   - `cd client && npm install`
   - `cd ../server && npm install`
3. Start the project:
   - `npm start` (Client)
   - `node index.js` (Server)
